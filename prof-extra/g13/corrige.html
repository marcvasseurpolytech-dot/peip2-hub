Quantifiers — Les bonnes nuances

📌 Rappel grammatical

some (affirmatif, propositions) / any (négatif, interrogatif) + composés : someone/anybody/nowhere…

each (un par un, insiste sur l'individu) vs every (ensemble global, sens proche de "tous").

one … the other (2 éléments) vs some … others (plusieurs éléments, plusieurs sous-groupes).

little / few = peu (connotation négative, manque) vs a little / a few = un peu (connotation positive, suffisant) — little avec indénombrable, few avec dénombrable.
