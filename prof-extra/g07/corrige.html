Conditionals — Les 3 types (+ le conditionnel passé)

📌 Rappel grammatical

Type 1 (réel/probable, futur) : If + present simple, ... will + V
If it rains, I will stay home.

Type 2 (irréel présent/futur) : If + past simple, ... would + V (were pour to be, toutes personnes)
If I were you, I would apologise.

Type 3 — le conditionnel passé (irréel passé, regret) : If + past perfect (had + V3), ... would have + V3
If I had studied more, I would have passed the exam.
→ décrit une situation qui ne s'est pas réalisée dans le passé : ici, l'étudiant n'a pas assez étudié et n'a donc pas réussi.

Type mixte : condition passée → conséquence présente : If + past perfect, ... would + V
If I had taken the job, I would be rich now.
